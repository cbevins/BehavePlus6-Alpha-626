Crown Fill Portion

Crown fill portion (percent or fraction) is an intermediate value in calculating wind adjustment factor (WAF). It is calculated from canopy cover, canopy height, and crown ratio. If crown fill portion is less than 5%, the calculation for unsheltered WAF is used; otherwise, the calculation is for sheltered WAF. If WAF is an input, the value for crown fill portion is not meaningful and is set to zero.

I/O Module If Notes
Input None    
Output SURFACE If Wind speed is entered as 20-ft (or 10-m) wind and Calculated wind adjustment factor is selected as an input option.
