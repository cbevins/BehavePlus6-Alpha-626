Application Appearance Options

This page controls the appearance of the picture pane, help pane, and page border in all the BehavePlus documents that are open in the application.

The options you set on this tab are preserved between BehavePlus sessions.

Appearance Controls

  • Show help pane
    • Checking this box causes an embedded help browser to appear in the right pane of every BehavePlus dialog as it is displayed. The help pane initially contains information describing the current dialog, but can also be used to link to all the other BehavePlus documentation in the program.
    • Unchecking this box hides the help pane, yielding more space to the dialog's central, context-specific pane.
    • The state of this control determines the appearance of every dialog as it is first displayed. The Help check box at the bottom of every BehavePlus dialog overrides the application-wide setting to dynamically show or hide the help pane for that particular dialog only.

  • Show picture pane
    • Checking this box causes a picture to appear in the left pane of every BehavePlus dialog as it is displayed.
    • Unchecking this box hides the picture, yielding more space to the dialog's central, context-specific pane.
    • The state of this control determines the appearance of every dialog as it is first displayed. The Picture check box at the bottom of every BehavePlus dialog overrides the application-wide setting to dynamically show or hide the picture pane for that particular dialog only.

  • Border Color
    • This combo box specifies the color of the border line that appears around each BehavePlus page. The default color is black.

  • Border Width
    • This spin box specifies the width of the border line that appears around each BehavePlus page. Border width ranges from 1 to 9. A width of 0 results in no border line. The default is a width of 1.
