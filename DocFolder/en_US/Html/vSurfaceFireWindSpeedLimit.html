Effective Wind Speed Limit

The effective wind speed limit is a function of reaction intensity (Rothermel 1972). Andrews, Cruz, and Rothermel (2013) reanalyzed the original data as well as more recent data to conclude that the wind limit is too restrictive. They recommend that the wind limit not be imposed. The user has the option to Impose maximum effective wind speed limit? (Yes or No). If Yes, the effective wind speed limit is used in the calculations. If No, effective wind speed from input wind and slope values are used. The output variable Maximum Effective Wind Exceeded? is available whether the wind limit is imposed or not.
