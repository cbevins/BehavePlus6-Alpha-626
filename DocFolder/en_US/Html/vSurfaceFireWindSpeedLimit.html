Surface Fire Effective Wind Speed Limit

The surface fire effective wind speed limit is the wind speed above which the predicted rate of spread is assumed to be constant. It is a function of reaction intensity (Rothermel 1972). Andrews, Cruz, and Rothermel (2013) reanalyzed the original data as well as more recent data to conclude that the current wind limit is too restrictive. They recommend that the wind limit NOT be imposed. In the rare case where head fire rate of spread is greater than effective wind speed (for effective wind > 0), surface fire rate of spread is set to the effective wind speed.

All of the spatial fire behavior modeling systems using the Rothermel fire spread model currently impose the wind limit. For consistency, therefore, the example worksheets continue to impose the wind limit. This option can be easily changed in the SURFACE module input options for wind speed. The user has the option to Impose maximum effective wind speed limit? (Yes or No). If Yes, the effective wind speed limit is used in the calculations. If No, effective wind speed from input wind and slope values are used. The output variable Surface Fire Maximum Effective Wind Exceeded? is available whether the wind limit is imposed or not.
