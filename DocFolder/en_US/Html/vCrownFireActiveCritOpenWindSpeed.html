Active Crown Critical Open Wind Speed

The active crown critical open wind speed is an intermediate value that represents the critical open wind speed required for sustaining a fully active crown fire (Scott and Reinhardt 2001; p. 19, eqn. 20). It is used to calculate the critical surface fire rate of spread required for crown fire initiation in the Scott and Reinhardt (2001) crown fire method.

I/O Module If Notes
Input None    
Output CROWN If Crown fire is calculated using Scott and Reinhardt (2001) is selected as an input option. Intermediate value
