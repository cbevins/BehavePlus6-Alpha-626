Chaparral Fuel Type

Chaparral fuel type is applicable only for the special case chaparral fuel model (Rothermel and Philpot 1973; Cohen 1986). This dynamic fuel model recognizes two distinct chaparral fuel types.

  • chamise
  • mixed brush

Chaparral fuel type is used, along with chaparral fuel bed depth to determine chaparral total fuel load.

I/O Module If Notes
Input SURFACE If Fuel is entered as special case fuel model, chaparral and Total fuel load is specified on the worksheet are selected as input options.  
Output None    

For more information on chaparral fuel model parameters, see Chaparral Details.
