Fire Name

The fire name is the name of the fire or project for which the projection is being created. This is one of the lines added when Fire behavior projection worksheet header is selected.

Other lines can be added to the worksheet header by selecting Configure > Appearance preferences > Worksheet and checking the Prescribed fire plan worksheet header and Training worksheet header check boxes.

Additional description can be added in the Notes section of the worksheet.
