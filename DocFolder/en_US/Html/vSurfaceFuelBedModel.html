Fuel Model

A fuel model is a set of numerical values that describes the fuel inputs for Rothermel's (1972) mathematical model to predict surface fire spread.

There are 53 standard fire behavior fuel models including the original 13 described by Anderson (1982), plus the 40 defined by Scott and Burgan (2005). Custom fuel models can be developed and saved for later use. Standard and custom fuel models can be used with one of the three options for calculating a weighted surface rate of spread through two fuel models. BehavePlus also offers special case fuel models that require different input: palmetto-gallberry (Hough and Albini 1978) and western aspen (Brown and Simmerman 1986).

To view the parameters of a particular fuel model listed under the Guide button, right-click on its name and select View parameters from the menu.

Information on Standard and Custom fuel models can be found below.

Information on using two fuel models can be found in Two Fuel Model Options.

Descriptions and photographs for the 13 original standard fire behavior models are taken from Anderson (1982). Descriptions and photographs of the 40 standard fuel models are taken from Scott and Burgan (2005).

Custom Fuel Models

Custom fuel models can be developed and used within BehavePlus. To create a custom fuel model in BehavePlus, it is best to use the example worksheets specifically designed for fuel modeling: FuelModeling.bpw (English units) and FuelModelingMetric.bpw (metric units) by going to File > Open worksheet and looking in the ExampleWorksheets folder. These worksheets allow greater precision in input data. You can also click on Configure > Module selection and press the Surface Fire Spread (SURFACE) Options button. Under the Fuel tab, activate the radio button titled Fuel is entered as... fuel parameters. The resulting input worksheet requests all the parameters needed to define a custom fuel model.

Fuel model parameters that can be varied are:

  • fuel load for each fuel particle diameter size class, live and dead
  • surface area-to-volume ratio for each fuel particle diameter size class, live and dead
  • fuel bed depth
  • heat content of the dead and live fuel
  • dead fuel moisture content of extinction

To save a custom fuel model, press File > Save as a fuel model > BehavePlus format.

By default only the 53 standard fire behavior fuel models appear in the choices list under the Guide button. Custom fuel models do not appear in the list until the folder containing them has been attached via Configure > Fuel model set selection. The attached fuel models will then be accepted as valid inputs to the worksheet and will appear in the list under the Input Guide window. Custom fuel models must be attached with every new session of BehavePlus.

Once a custom fuel model has been saved in BehavePlus format, it can be saved in a format that can be read by NEXUS, FlamMap and FARSITE using File > Save as a fuel model > FARSITE format, English or File > Save as a fuel model > FARSITE format, metric. More information can be found in Fuel Model Export.
