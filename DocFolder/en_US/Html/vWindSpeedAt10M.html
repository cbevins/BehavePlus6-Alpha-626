10-meter Wind Speed

10-meter wind speed is measured 10 meters (32.8 feet) above the vegetation. The 10-meter wind speed is reduced to 20-foot wind speed in BehavePlus by dividing the 10-meter wind speed by 1.15; this is based on the logarithmic curve of wind speed described by Albini and Baughman (1979). For fire spread and intensity calculations in SURFACE, the 20-foot wind speed is then further reduced to midflame wind speed according to the wind adjustment factor.

If the SURFACE and SPOT or CROWN modules are active, and if Wind speed is entered as midflame height is selected as an input option for SURFACE, there is a conflict because the wind speed required for SPOT or CROWN is at the 20-foot height. The 10-meter wind speed is automatically adjusted to 20-foot wind speed in this situation. The program offers a choice of how to resolve the conflict.

I/O Module If Notes
Input SURFACE If Wind speed is entered as 10-m wind and Input (or Calculated) wind adjustment factor is selected as an input option.  
CROWN If Wind speed is entered as 10-m wind and Input (or Calculated) wind adjustment factor is selected as an input option in SURFACE. Otherwise, 20-foot wind speed is requested
SPOT If Wind speed is entered as 10-m wind and Input (or Calculated) wind adjustment factor is selected as an input option in SURFACE. Otherwise, 20-foot wind speed is requested
SCORCH If Wind speed is entered as 10-m wind and Input (or Calculated) wind adjustment factor is selected as an input option in SURFACE.  
Output None
