Passive Crown Fire Flame Length

Passive crown fire flame length, based on 20-foot wind speed, is calculated from passive crown fireline intensity as described by Scott and Reinhardt (2001). The flame length equation is from Thomas (1963) and differs from that used for surface fire (Byram 1959).

Passive crown fire outputs are only valid for the Scott and Reinhardt (2001) crown fire method.

I/O Module If Notes
Input None    
Output CROWN If Crown fire is calculated using Scott and Reinhardt (2001) is selected as an input option.
