Prescribed Fire Name

The prescribed fire name identifies the prescribed fire. This is one of the lines added when Prescribed fire plan worksheet header is selected.

Other lines can be added to the worksheet header by selecting Configure > Appearance preferences > Worksheet and checking the Training worksheet header and Fire behavior projection worksheet header check boxes.

Additional description can be added in the Notes section of the worksheet.
