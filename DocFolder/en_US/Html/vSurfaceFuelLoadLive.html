Total Live Fuel Load

Total live fuel load is the sum of the load for all live fuel classes. For standard and custom fuel models this includes both live woody and live herbaceous fuel load. The calculation is made after the load transfer is done for dynamic fuel models. This variable applies only to dynamic fuel models.

See the Fuel Model Type for specifics about the load transfer equation.

I/O Module If Notes
Input None    
Output SURFACE   Of primary interest for dynamic fuel models
