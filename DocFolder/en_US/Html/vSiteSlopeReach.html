Slope Horizontal Distance

Slope horizontal distance is the distance between two points on the ground, and is also known as the run. This is an intermediate value in the calculation of slope steepness from map measurements.

I/O Module If Notes
Input None    
Output SURFACE If Slope steepness is calculated from map measurements is selected as an input option.  

Calculating Slope Steepness

  • Assume the Slope Elevation Change (rise) is 400 ft.
  • Assume the Slope Horizontal Distance (run) is 800 ft.
  • Slope Steepness = (400 / 800) × 100% = 50%, or
  • Slope Steepness = arctan(0.50) = 27 degrees
