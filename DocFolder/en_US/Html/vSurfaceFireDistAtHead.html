Surface Fire Heading Spread Distance

The surface fire heading spread distance is the distance the head surface fire travels in a specified period of time. This is the same as surface fire spread distance in the SURFACE module for a heading fire.

For more information on fire spread direction options, see Fire Spread Directions.


I/O Module If Notes
Input None    
Output SIZE   Can also be output as a map distance if Display output distances in map units is checked.
