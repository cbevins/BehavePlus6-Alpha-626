Live Woody Fuel Load

Live woody fuel load is the weight of live woody fuel per unit area and is a fuel model parameter. Live woody fuels include the foliage and very fine stems of living shrubs.

Live woody fuels are perennial and frequently have surface area-to-volume ratios ranging from 1,000 to 2,000 ft2 / ft3 (3,281 to 6,562 m2 / m3).

I/O Module If Notes
Input SURFACE If Fuel is entered as fuel parameters is selected as an input option. If Fuel is entered as fuel models or a special case fuel model is selected then the appropriate stored value is used.
Output SURFACE If Fuel is entered as special case fuel model, western aspen is selected as an input option.
