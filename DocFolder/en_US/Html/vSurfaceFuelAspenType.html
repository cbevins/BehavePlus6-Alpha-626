Aspen Fuel Model

There are five special case aspen fuel models (Brown and Simmerman 1986). Their use requires different user input than is required for standard or custom fuel models (Brown and Simmerman 1986).

Aspen Fuel Model Inputs
Aspen Fuel Model Fuel Bed Depth 10-h Fuel Load
  ft m tons / ac tonnes / ha
Aspen/shrub 0.65 0.20 0.975 0.297
Aspen/tall forb 0.30 0.09 0.475 0.145
Aspen/low forb 0.18 0.05 1.035 0.315
Mixed/shrub 0.50 0.15 1.340 0.408
Mixed forb 0.18 0.05 1.115 0.340

For all aspen fuel models, the following are constant:

  • Live and dead heat content = 8,000 Btu / lb (18,622 kJ / kg)
  • Moisture of extinction = 25%
  • 10-h SA/V = 109 ft2 / ft3 (358 m2 / m3)
  • Live herbaceous SA/V = 2,800ft2 / ft3 (9,186 m2 / m3)

Aspen curing level is used to calculate the following for each of the five aspen fuel models:

  • 1-h fuel load
  • Live herbaceous fuel load
  • Live woody fuel load
  • 1-h SA/V
  • Live woody SA/V

For more information on aspen fuel parameters, see Western Aspen Details.
