Aspen Fuel Model

There are five special case western aspen fuel models (Brown and Simmerman 1986). Their use requires different user inputs than are required for standard or custom fuel models (Brown and Simmerman 1986), which resulted in developing the special case model.

I/O Module If Notes
Input SURFACE If Fuel is entered as special case fuel model, western aspen is selected as an input option.  
Output None

The following values are constant for all western aspen fuel models.

  • Live and dead fuel heat content = 8,000 Btu / lb (18,622 kJ / kg)
  • Moisture of extinction = 25%
  • 10-h fuel surface area-to-volume ratio = 109 ft2 / ft3 (358 m2 / m3)
  • Live herbaceous surface area-to-volume = 2,800ft2 / ft3 (9,186 m2 / m3)

Aspen curing level is used to calculate the following for each of the five aspen fuel models.

  • 1-h fuel load
  • 1-h fuel surface area-to-volume ratio
  • Live herbaceous fuel load
  • Live woody fuel load
  • Live woody fuel surface area-to-volume ratio

For more information on aspen fuel parameters, see Western Aspen Details.
