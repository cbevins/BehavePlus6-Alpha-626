Relative Packing Ratio

Relative packing ratio is the ratio of the fuel bed packing ratio to the optimum packing ratio (Beta divided by Beta-op). It is a factor in determining the optimum reaction velocity. This is an intermediate value in Rothermel's (1972) surface fire spread model.

I/O Module If Notes
Input None    
Output SURFACE   Intermediate value
