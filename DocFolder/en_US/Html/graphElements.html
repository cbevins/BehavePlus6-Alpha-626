Graph Element Options

This page controls the color and width of BehavePlus output graph elements.

Appearance Controls

  • Background Color
    Controls the graph's background color. Select one of the predefined colors from the drop-down list.


  • Rainbow Colors
    Controls the number of colors used to display graph lines or bars when Rainbow coloring is selected.

    If 4 colors are requested, for example, 4 equally-spaced hues from the color spectrum are used to display each graph line or bar in turn. If there are more than 4 curves, then the colors are reused.


  • Bar Color
    Controls the color used to fill graph bars. Select one of the pre-defined colors from the drop-down list.

    The special Rainbow color cycles through a series of colors so that each bar is filled with its own color. The number of rainbow colors is set by the Rainbow Colors control.


  • Curve Points
    Controls the number of points calculated for each curve in the graph. A straight line segment is drawn between each pair of points.

  • Curve Color
    Controls the color used to draw graph lines. Select one of the pre-defined colors from the drop-down list.

    The special Rainbow color cycles through a series of colors so that each line is drawn with its own color. The number of rainbow colors is set by the Rainbow Colors control.


  • Curve Width
    Controls the width of graph lines.


  • Axis Color
    Controls the color used to draw graph axis lines, tic marks, and axis labels. Select one of the pre-defined colors from the drop-down list.


  • Axis Width
    Controls the width of axis lines drawn on the graph.


  • Gridline Color
    Controls the color used to draw graph grid lines. Select one of the pre-defined colors from the drop-down list.


  • Gridline Width
    Controls the width of grid lines drawn on the graph. Select None to prevent the display of any grid lines.
