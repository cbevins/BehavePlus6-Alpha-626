Power Ratio

Power ratio is power of the fire divided by power of the wind (Rothermel 1991). If the power ratio is less than one (power of the wind is greater than power of the fire), the active crown fire is estimated to be wind-driven. The power ratio provides more information than the Yes or No for the wind-driven crown fire? variable. It is instructive to know whether the power ratio is close to or very different from 1. A value much greater than one may indicate increased confidence in model prediction of wind-driven crown fires. As always, expert knowledge should be used to confirm model predictions.

Fire Type outputs are only valid for the Rothermel (1991) crown fire method.

I/O Module If Notes
Input None    
Output CROWN If Crown fire is calculated using Rothermel (1991) is selected as an input option.
