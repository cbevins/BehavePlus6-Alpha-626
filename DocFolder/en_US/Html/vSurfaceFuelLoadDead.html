Total Dead Fuel Load

Total dead fuel load is the sum of the load of all dead fuel classes. For standard and custom fuel models this includes 1-h, 10-h, 100-h, and dead herbaceous fuel. This calculation is done after the load transfer is done for dynamic fuel models. This variable is most meaningful in an examination of dynamic fuel models.

I/O Module If Notes
Input None    
Output SURFACE   Of primary interest for dynamic fuel models
