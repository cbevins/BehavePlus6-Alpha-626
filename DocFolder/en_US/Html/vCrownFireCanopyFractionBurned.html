Canopy Fraction Burned

Canopy fraction burned is used to calculate passive crown fire rate of spread. It has a value of 0 for a surface fire and a value of 1 for a fully active crown fire (Scott and Reinhardt 2001; p. 41, eqn. 28). The associated passive crown fire rate of spread will be somewhere between the surface fire rate of spread and the active crown fire rate of spread.

Passive crown fire outputs are only valid for the Scott and Reinhardt (2001) crown fire method.

I/O Module If Notes
Input None    
Output CROWN If Crown fire is calculated using Scott and Reinhardt (2001) is selected as an input option.
