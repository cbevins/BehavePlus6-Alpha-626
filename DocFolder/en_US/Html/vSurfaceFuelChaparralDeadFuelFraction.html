Chaparral Fuel Dead Load Fraction

Chaparral fuel dead load fraction is the percent or fraction of the total fuel load that is dead. It is a required input to the chaparral special case fuel model (Rothermel and Philpot 1973).

The chaparral fuel dead load fraction is used to divide the total fuel load into dead and live fuel load categories. This special case chaparral fuel model differs from fuel model 4 (described by Anderson 1982).

I/O Module If Notes
Input SURFACE If Fuel is entered as special case fuel model, chaparral is selected as an input option.  
Output None    

For more information on chaparral fuel model parameters, see Chaparral Details.
