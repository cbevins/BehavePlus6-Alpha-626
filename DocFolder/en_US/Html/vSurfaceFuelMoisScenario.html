Moisture Scenario

A fuel moisture scenario is a set of fuel moisture values representing moisture condition of the surface fuel: 1-h, 10-h, 100-h, live herbaceous, and live woody fuel moisture. Fuel moisture scenarios are analogous to fuel models; they package a number of parameters into a labeled set. You can also define and save a moisture scenario for later use.

The moisture scenarios included with BehavePlus were used by Scott and Burgan (2005) when developing their standard fuel models. Fuel moisture scenarios are also useful for testing custom fuel models or for developing fire prescriptions. For example, a moisture scenario might be developed to represent a location's 90th, 95th, and 97th percentile fire weather conditions or to represent the fuel moisture conditions for several specific fires.

In addition, since BehavePlus only allows two variables to change at a time, users can explore more variations in moisture using scenarios. For example, users change both the 1-h and 10-h fuel moisture in the moisture scenario and then also examine the impact of wind speed in addition to those variations. Without using moisture scenarios, the number of runs needed to test all of the different options would increase. All of the various scenarios can be entered on the input worksheet to generate comparative tables and graphs (not possible using fuel moisture by size class).

Using moisture scenarios

  • To use a fuel moisture scenario, enter its name in the input field or press the Guide button to select scenarios from the list.
  • Custom moisture scenarios can be developed and used within BehavePlus. To create a custom moisture scenario,
    1. Click on Configure > Module selection and press the Surface Fire Spread (SURFACE) Options button.
    2. Under the Input Options > Moisture tab, activate the radio button titled Moisture is entered by... individual size class.
    3. The resulting input worksheet requests all the parameters needed to define a custom moisture scenario.
    4. To save the values as a custom moisture scenario, press File > Save As a moisture scenario.
  • Before custom moisture scenarios can be used by BehavePlus, the folder containing the scenarios must be attached to the run via Configure > Moisture scenario set selection. The attached moisture scenarios will then be accepted as valid inputs to the worksheet and will appear in the list under the Guide button.
  • To view the parameters of a particular moisture scenario, right-click on its name and select View parameters from the context menu.
