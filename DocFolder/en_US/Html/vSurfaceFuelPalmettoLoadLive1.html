P-G Live Fine Fuel Load

The P-G live fine fuel load for the palmetto-gallberry fuel model is the fuel load of live palmetto-gallberry stems with 0- to 0.25-inch (0 to 0.64 cm) diameters. This variable is calculated from P-G age of rough and P-G height of understory.

Module

If

Notes
Input None    
Output SURFACE If Fuel is entered as special case fuel model, palmetto-gallberry is selected.  

For more information on palmetto-gallberry fuel parameters, see Palmetto-Gallberry Details.
