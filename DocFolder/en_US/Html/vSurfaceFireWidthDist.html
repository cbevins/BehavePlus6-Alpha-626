Surface Fire Maximum Width

The surface fire maximum width is based on the elliptical fire shape model and is maximum width of the ellipse. Surface fire flanking spread distance is half the maximum fire width.

I/O Module If Notes
Input None    
Output SIZE   Can also be output as a map distance if Display output distances in map units is checked.
