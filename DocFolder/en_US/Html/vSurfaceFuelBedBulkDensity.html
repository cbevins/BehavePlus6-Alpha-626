Bulk Density

Bulk density (lb/ft3) for a fuel bed is the total oven-dry fuel loading per cubic volume of fuel bed (fuel loading divided by fuel bed depth). This is an intermediate value in Rothermel's (1972) surface fire spread model.

I/O Module If Notes
Input None    
Output SURFACE   Intermediate value
