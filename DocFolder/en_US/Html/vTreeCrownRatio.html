Crown Ratio

Crown ratio is the proportion of the total vertical tree height that is occupied by the vertical length of the tree crown. A crown ratio of 0.5 indicates that half of the total vertical tree height is covered in branches and leaves. Crown ratio and canopy height are used to find canopy base height, which is used to calculate tree crown length scorched in the MORTALITY module. Crown ratio is also used in the SURFACE module to calculate wind adjustment factor.

I/O Module If Notes
Input MORTALITY If the CROWN module is not also selected.  
SURFACE If the CROWN module is not also selected, and if Wind speed is entered as 20-ft (or 10-m) wind and calculated wind adjustment factor is selected as an input option.  
Output SURFACE If the CROWN module is also selected, and if Wind speed is entered as 20-ft (or 10-m) wind and calculated wind adjustment factor is selected as an input option.
