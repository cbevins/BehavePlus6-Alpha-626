Resource Name

Resource name is a simple, one-word identifier for a containment resource. If Suppression input entered for multiple resources is selected, the resource name is used to associate the resource with its resource arrival time, resource line production rate, and resource duration. For example, Resource #3 will be associated only with the 3rd value in each of the lists. The same suppression tactic and line construction offset will be used for all resources. The number of resource name entries on the worksheet must match the number of resource arrival time entries, resource line production rate entries, and resource duration entries.

The resource name must be a single word with no embedded blanks or tabs. Hyphens can be used. Examples include: Engine1, Type1Crew1, Type_2_Crew_2, and Bulldozer-1.

I/O Module If Notes
Input CONTAIN If Suppression input entered for multiple resources is selected as an input option.  
Output None
