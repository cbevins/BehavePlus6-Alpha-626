Resource Name

Resource name is a simple, one-word identifier for a containment resource. If Suppression input entered for multiple resources is selected, the resource name is used to associate the resource with its resource arrival time, resource line production rate, and resource duration. The resource name must be a single word with no embedded blanks or tabs. Examples include: Engine1, Type1Crew1, Type_2_Crew_2, and Bulldozer1.

The number of resource name entries on the worksheet must match the number of resource arrival time entries, resource line production rate entries, and resource duration entries.
