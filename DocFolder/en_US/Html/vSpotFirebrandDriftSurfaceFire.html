Firebrand Drift Distance from a Wind-Driven Surface Fire

Firebrand drift distance from a wind-driven surface fire is the distance that the firebrand is carried as it is being lofted. This is an intermediate value in the calculations useful for examining the spotting distance model.

I/O Module If Notes
Input None    
Output SPOT   Intermediate value
