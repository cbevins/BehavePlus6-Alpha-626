WAF Calculation

WAF (Wind Adjustment Factor) Calculation indicates the method of calculating wind adjustment factor (WAF). If WAF is input rather than calculated, this variable is not meaningful and is set to Input.

The variable WAF Calculation has three possible values.

  • Sheltered
  • Unsheltered
  • Input

If the crown fill portion is less than 5%, the Unsheltered model is used; otherwise the Sheltered model is used.

I/O Module If Notes
Input None    
Output SURFACE If Wind speed is entered as 20-ft (or 10-m) wind and Calculated wind adjustment factor is selected as an input option.
