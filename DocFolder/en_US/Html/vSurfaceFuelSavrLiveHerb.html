Live Herbaceous Surface Area-to-Volume Ratio

Live herbaceous surface area-to-volume ratio (SA/V) is the total surface area of a herbaceous plant leaf divided by the volume occupied by the plant leaf. Live herbaceous surface area-to-volume ratio is a fuel model parameter.

I/O

Module

If

Notes
Input SURFACE If Fuel is entered as fuel parameters is selected as an input option. If Fuel is entered as fuel models then the value stored for the standard or custom fuel model is used.
Output None
