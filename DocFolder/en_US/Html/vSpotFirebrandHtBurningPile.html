Firebrand Height from a Burning Pile

Firebrand height from a burning pile is the maximum height to which a firebrand is lifted and is still viable for starting a fire when it lands. This is an intermediate value useful for examining the model for spotting distance from a burning pile.

I/O Module If Notes
Input None    
Output SPOT   Intermediate value
