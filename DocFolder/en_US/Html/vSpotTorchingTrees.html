Number of Torching Trees

Number of torching trees is the number of trees that torch simultaneously during group tree torching. It is not the number of trees torching individually across a landscape. This value is used to calculate the height to which firebrands are lofted.

I/O Module If Notes
Input SPOT If Spotting Distance from Torching Trees is selected as an output variable.  
Output None
