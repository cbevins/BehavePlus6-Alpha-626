CONTAIN Input Options

This page controls the CONTAIN Module's input options.

If suppression input is entered for a single resource, multiple values can be entered for input variables (such as production rate) to produce tables and graphs. For example, final fire size can be calculated for a range of line production rates.

If suppression input is entered for multiple resources, you have control over the sequencing and arrival times of all resources. Each resource is dispatched in turn and arrives to produce line at the specified time. For example, if four engines or crews are scheduled for dispatch at hourly intervals, but the fire is contained after three hours, only the first three resources are included in the production and cost results.

Options

Suppression input is entered for....
  a single resource. This is a simplification of the containment modeling capabilities. This case allows multiple values to be entered for the variables associated with the single resource: line production rate, arrival time, duration, base cost, and/or hourly cost. Tables and graphs can be produced for ranges.
  multiple resources. This option allows multiple resources to suppress a fire. Each resource (as identified by its Resource Name) is assigned a single value for its associated variables: line production rate, arrival time, duration, base cost, and hourly cost. This option allows you to control the sequencing and arrival times of the resources.
