Module Selection

Check those Modules you wish to activate. Press a Module's Options... button to configure input options or change the set of output variables. Indentation indicates modules that can be linked together, with the output from one used as input to the next.

Additional Options

Checking the box...
  Display output distances in map units adds map descriptors to the worksheet so output variables such as Surface Fire Spread Distance are displayed in both real world and map units.
Table shading for acceptable fire conditions adds a section to the worksheet where values for acceptable conditions can be entered (for example, Surface Fire Flame Length of 2 to 4 ft (0.6 to 1.2 m)).
