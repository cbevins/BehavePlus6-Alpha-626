Passive Crown Fire Spread Distance

The passive crown fire spread distance is the head fire (forward) spread distance of a passive crown fire (surface fire with torching trees) calculated from passive crown fire rate of spread and elapsed time. It is the overall spread distance for a sustained run of passive crown fire activity over several hours. This spread distance implicitly includes the effects of spotting.

Passive crown fire outputs are only valid for the Scott and Reinhardt (2001) crown fire method.

I/O Module If Notes
Input None    
Output CROWN If Crown fire is calculated using Scott and Reinhardt (2001) is selected as an input option. Can also be output as a map distance if Display output distances in map units is checked.
