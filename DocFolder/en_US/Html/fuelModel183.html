Fire Behavior Fuel Model TL3 (183) -- Moderate Load Conifer Litter

Scott and Burgan (2005) classify Fire Behavior Fuel Model TL3 (183) in the Timber Litter Fuel Type and describe it as follows:

The primary carrier of fire in TL3 is moderate load conifer litter, light load of coarse fuels. Spread rate is very low; flame length low.

Fuel model parameters are available by viewing the Fuel Model Table of Parameters or right-clicking on the fuel model description in the Input Guide window and selecting View parameters.

Scott and Burgan's (2005) photographs on page 59 are examples of fuels fitting this model.
