Western Aspen Details

The Western Aspen fuel option in BehavePlus implements the fuel modeling from Brown and Simmerman (1986) and the aspen mortality model from Brown and DeByle (1987). Probability of aspen mortality is included with this option rather than in the MORTALITY module because it is a different model with different input requirements. Flame length, DBH, and fire severity (low or moderate+) are used to calculate aspen mortality.

Following are constants for all aspen fuel models:

  • Live and dead heat content = 8,000 Btu / lb (18,622 kJ / kg)
  • Moisture of extinction = 25%
  • 10-h SA/V = 109 ft2 / ft3 (358 m2 / m3)
  • Live herbaceous SA/V = 2,800ft2 / ft3 (9,186 m2 / m3)

Aspen curing level is used to calculate the following for each of the 5 aspen fuel models:

  • 1-h fuel load
  • Live herbaceous fuel load
  • Live woody fuel load
  • 1-h SA/V
  • Live woody SA/V
