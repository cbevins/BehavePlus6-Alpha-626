BehavePlus Help

Users of the BehavePlus fire modeling system are expected to have prior training and experience with fire behavior modeling. A user is expected to have basic knowledge of fire, fuels, and influencing factors.

  • The Features Paper (PDF) is available from the Help menu. This documents BehavePlus design and features to help users better understand the system and use it more effectively.
  • A Variable Help (PDF) document is available from the Help menu. It includes input and output tables and a description of each variable in the program.
  • A Help window is available during program operation to provide context-sensitive help and contains more information than the variable help document.
  • A Guide button next to each worksheet entry provides a description of the variable.
  • Holding the cursor over variables in the output selection list provides a description of the variable.
  • A Master Index of variables, figures, tables, and guides is available from the Help menu and from the Help window.
  • Tutorials and other supporting documentation, including publications, are available on www.frames.gov/behaveplus.
