Flat Terrain Spotting Distance from Active Crown Fire

Flat terrain spotting distance from an active crown fire is the maximum spotting distance over flat terrain. This value is then adjusted for terrain. This is an intermediate value in the calculations useful for examining the model for spotting distance from active crown fire.
