Active Crown Fire Spread Distance

The active crown fire spread distance is the head fire (forward) spread distance of an active crown fire calculated from active crown fire rate of spread and elapsed time. It is the overall spread distance for a sustained run over several hours. This spread distance includes the effects of spotting since spotting is implicitly included in Rothermel's (1991) method for calculating active crown fire rate of spread.

I/O Module If Notes
Input None    
Output CROWN   Can also be output as a map distance if Display output distances in map units is checked.
