Chaparral Total Fuel Load

Chaparral total fuel load is the total amount of fuel in the chaparral and is a required input to the chaparral special case fuel model (Rothermel and Philpot 1973; Cohen 1986). It can either be entered directly or calculated from chaparral type and fuel bed depth.

The chaparral total fuel load is divided into dead and live fuel load categories using the chaparral dead fuel load fraction.

I/O Module If Notes
Input SURFACE If Fuel is entered as special case fuel model, chaparral and Total fuel load is specified on the worksheet are selected as input options.  
Output SURFACE If Fuel is entered as special case fuel model, chaparral and Total fuel load is calculated from fuel bed depth and type are selected as input options.  

For more information on chaparral fuel model parameters, see Chaparral Details.
