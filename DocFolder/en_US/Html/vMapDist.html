Map Distance

The map distance is the distance between two points on a map generally expressed in inches or centimeters. It is used in the calculation of slope steepness.

I/O

Module

If

Notes
Input SURFACE

If Slope steepness is calculated from map measurements is selected as an input option.

  
Output None    

If Display output distances in map units is checked in the Module Selection dialog, then appropriate output distances are also shown as map distances for the following variables.

  • SURFACE
    • Surface Fire Spread
  • CROWN
    • Crown Spread Spread
  • SIZE
    • Backing Spread Distance
    • Heading Spread Distance
    • Fire Length
    • Flanking Spread Distance
    • Maximum Fire Width
  • SPOT
    • Spotting Distance from a Burning Pile
    • Spotting Distance from Active Crown Fire
    • Spotting Distance from Torching Trees
    • Spottting Distance from Wind-Driven Surface Fire
