Map Distance

The map distance is the distance between two points on a map and is generally expressed in inches or centimeters. It is used in the calculation of slope steepness.

I/O Module If Notes
Input SURFACE If Slope steepness is calculated from map measurements is selected as an input option.  
Output None    

If Display output distances in map units is checked in the Module Selection dialog, then appropriate output distances are also shown as map distances for the following variables.

  • SURFACE
    • Surface Fire Spread Distance
  • CROWN
    • Active Crown Spread Distance
    • Passive Crown Spread Distance
  • SIZE
    • Surface Fire Backing Spread Distance
    • Surface Fire Flanking Spread Distance
    • Surface Fire Heading Spread Distance
    • Surface Fire Length
    • Surface Fire Maximum Width
  • SPOT
    • Spotting Distance from a Burning Pile
    • Spotting Distance from a Wind-Driven Surface Fire
    • Spotting Distance from Active Crown Fire
    • Spotting Distance from Torching Trees
