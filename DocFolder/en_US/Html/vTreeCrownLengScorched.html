Tree Crown Length Scorched

Tree crown length scorched is the vertical amount of the crown that has been scorched. It is used to calculate percent tree crown volume scorched, which is then used to calculate tree mortality for some species. Tree crown length scorched is calculated as follows.

  • Tree crown length scorched = Scorch height - (Canopy height - Live crown length).

If predicted scorch height is greater than the canopy height, then tree crown length scorched is set equal to the tree crown length.

I/O Module If Notes
Input None    
Output MORTALITY
