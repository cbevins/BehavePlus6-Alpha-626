Crown Fuel Load

Crown fuel load is an intermediate value that is used to calculate canopy heat per unit area as described by Rothermel (1991). Crown fuel load is found by multiplying canopy bulk density by the difference between canopy height and canopy base height.

I/O Module If Notes
Input None    
Output CROWN   Intermediate value
