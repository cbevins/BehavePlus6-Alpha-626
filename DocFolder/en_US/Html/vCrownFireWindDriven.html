Wind-Driven Crown Fire?

Wind-driven crown fire? is either Yes or No. Yes indicates an active crown fire may be wind-driven (power of the wind is greater than the power of the fire). No indicates the active crown fire may be plume-dominated (power of the fire is greater than power of the wind). Consider the value of the power ratio in addition to a yes or no answer. As always, expert knowledge should be used to confirm model predictions.

Fire Type outputs are only valid for the Rothermel (1991) crown fire method.

I/O Module If Notes
Input None    
Output CROWN If Crown fire is calculated using Rothermel (1991) is selected as an input option.
