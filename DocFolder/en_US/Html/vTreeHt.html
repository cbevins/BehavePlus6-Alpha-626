Torching Tree Height

Torching tree height is the distance from the ground to the top of a tree. It is the height of torching trees used to calculate maximum spotting distance in the SPOT module.

Torching tree height may be different from canopy height and downwind canopy height. Canopy height is used to calculate wind adjustment factor, crown fire intensity, and tree mortality. Downwind canopy height is used to calculate maximum spotting distance in SPOT and is the mean cover height in the downwind direction as it affects the distance a firebrand is carried.

I/O Module If Notes
Input SPOT If Spotting distance from torching trees is selected as an output variable.  
Output None
