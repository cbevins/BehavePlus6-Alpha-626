Live Herbaceous Fuel Load Remainder

Live herbaceous fuel load is the weight of the live herbaceous fuel that remains after some portion of the load is transferred to the dead herbaceous fuel class. This variable only applies to dynamic standard and custom fuel models.

See the Fuel Model Type for specifics about the load transfer equation.

I/O Module If Notes
Input None    
Output SURFACE If the standard or custom fuel model is dynamic.
