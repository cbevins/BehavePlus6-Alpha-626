Live Herbaceous Fuel Load Remainder

Live herbaceous fuel load is the weight of that class of fuel that remains after some of the load is transferred to the dead herbaceous fuel class. This is a meaningful variable only for dynamic standard and custom fuel models.

I/O

Module

If

Notes
Input None    
Output SURFACE If the standard or custom fuel model is dynamic.
