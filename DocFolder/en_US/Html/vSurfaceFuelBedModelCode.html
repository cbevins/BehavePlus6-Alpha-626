Fuel Model Code

Fuel models are described in a three-part naming scheme, including

  • fuel model code, which is a 1-7 character identifier such as 1 or gr1; and
  • fuel model number, which is an integer in the range 1-256.
  • fuel model name, which is a longer description of the fuel model.

The fuel model code is a 1-7 character identifier that facilitates communication and can be used as an input to some fire modeling systems, including BehavePlus. The fuel model code and number are identical for the original 13 fuel models described by Anderson (1982). The Scott and Burgan (2005) fuel models are grouped by fire-carrying type, such as GR (grass) or TU (timber-understory) followed by a number. The higher the number, the greater the heat per unit area. Custom fuel models require a code and number so that they can be exported for use in the FARSITE format.
