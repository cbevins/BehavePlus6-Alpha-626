100-h Fuel Load

The 100-h fuel load is the weight of the 100-h dead fuel per unit area and is a fuel model parameter. The 100-h timelag fuel category includes fuels from 1 to 3 inches (2.54 to 7.62 cm) in diameter. Fuels larger than those in this category (for example 1000-h fuel) do not contribute to the surface rate of spread as calculated by Rothermel's (1972) surface fire spread model.

I/O Module If Notes
Input SURFACE If Fuel is entered as fuel parameters is selected as an input option. If Fuel is entered as fuel models or a special case fuel model then the appropriate stored value is used.
Output None
