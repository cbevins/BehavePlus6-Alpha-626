Fire & Effects Units

This page controls the selection of units of measure for all Fire Behavior and Effects variables. The number of decimal places displayed for output variables can be changed. The number of decimal places displayed for input variables is determined by the precision of user input, not by this page (a change made in version 5.0).

To apply a previously saved custom Units Set to any BehavePlus worksheet, choose Configure > Units set selection > Custom from the menu and select the desired file.
