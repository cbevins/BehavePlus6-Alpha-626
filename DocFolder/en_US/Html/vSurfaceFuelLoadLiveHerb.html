Live Herbaceous Fuel Load

Live herbaceous fuel load is the weight of that class of fuel per area and is a fuel model parameter. Live herbaceous fuels are living grasses and forbs, either annual or perennial. If the fuel model is dynamic, the initial live herbaceous load is stored with the fuel model. A portion of the live herbaceous fuel load is transferred to dead herbaceous fuel load. The portion of the load transferred is either specified by the user or calculated from live herbaceous moisture.

If Fuel is entered as fuel parameters is selected as an input option.

SURFACE

If the standard or custom fuel model is dynamic.

If Fuel is entered as special case fuel model, western aspen is selected as an input option.

If the fuel model is dynamic, the value used in the calculations is the initial value reduced according to the Fuel Load Transfer portion.
