Number of Resources Used

The number of resources used indicates those actually needed to contain the fire. It may be less than the number of resources specified on the worksheet if the fire is contained before all resources arrive.

I/O Module If Notes
Input None    
Output CONTAIN If Suppression input entered for multiple resources is selected as an input option.
