Palmetto-Gallberry Details

The Palmetto-Gallberry fuel option in BehavePlus allows entry of vegetation data for southern rough fuels. The equations of Hough and Albini (1978) are then used to create fuel inputs for live and dead fuel by size class to define surface fuel parameters.

The Palmetto-Gallberry option assumes the following constants when assigning fuel parameters.

Table 1: Palmetto-Gallberry Fuel Constants
Fuel Model Variable English Value Metric Value
extinction moisture 40 percent
fuel bed depth 2/3 height of understory
low heat content 8,300 Btu / lb 19,321 kJ / kg
particle density, live fuel 46 lb / ft3 737 kg/m3
particle density, dead fuel 30 lb / ft3 481 kg/m3
total ash content, fraction 0.030
silica-free ash, live fuel, fraction 0.015
silica-free ash, dead fuel, fraction 0.010
SA/V, foliage 2,000 ft2 / ft3 6,562 m2 / m3
SA/V, 0-0.25 inch diameter stems 350 ft2 / ft3 1,148 m2 / m3
SA/V, 0.25-1 inch diameter stems 140 ft2 / ft3 459 m2 / m3
