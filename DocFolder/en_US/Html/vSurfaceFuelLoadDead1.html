1-h Fuel Load

The 1-h fuel load is the weight of the 1-h dead fuel per unit area and is a fuel model parameter. The one-hour (1-h) timelag dead fuel category includes dead fuel from 0 to 0.25 inches (0 to 0.64 cm) in diameter. This includes needles, leaves, cured herbaceous plants and fine dead stems of plants. This value does not include the dead herbaceous fuel load that is transferred from the live fuel in dynamic fuel models. The 1-h fuel load is an important fuel parameter for Rothermel's (1972) fire spread model.

I/O Module If Notes
Input SURFACE If Fuel is entered as fuel parameters is selected as an input option. If Fuel is entered as fuel models or a special case fuel model then the appropriate stored value is used.
Output SURFACE If Fuel is entered as special case fuel model, western aspen is selected as an input option.
