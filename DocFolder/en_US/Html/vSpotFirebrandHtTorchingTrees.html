Firebrand Height from Torching Trees

The firebrand height from torching trees is the maximum height to which a firebrand is lifted and is still viable for starting a fire when it lands. This is an intermediate value in the calculations useful for examining the model for spotting distance from torching trees.

I/O Module If Notes
Input None    
Output SPOT   Intermediate value
