Spotting Distance from a Burning Pile

Spotting distance from a burning pile is the maximum distance that one can expect potential spot fires resulting from firebrands from the burning pile. Flame height from a burning pile is an input used to calculate the lofting strength of embers from the burning pile.

I/O Module If Notes

Input

None

 

  

Output

SPOT

 

Can also be output as a map distance if Display output distances in map units is checked.
