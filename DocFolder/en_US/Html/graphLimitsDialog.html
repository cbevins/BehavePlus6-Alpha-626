Graph Y Axis Limits

When calculating a run, you can specify the Y axis minimum and maximum values for any or all of the output graphs in this Graph Limits Dialog box.

BehavePlus normally determines minimum and maximum Y axis limits for each graph. These recommended values are initially displayed in the Y Axis Minimum and Y Axis Maximum entry fields. You may leave these as is, or modify them, for example, to produce a series of graphs at the same scale.

Press "Ok" to use the Minimum and Maximum values displayed in the entry fields. Press "Cancel" to use the predetermined, recommended limits. Either way the result graphs will be displayed.
