Graph Y Axis Limits

You can control how graphs look in BehavePlus. In this window, you can specify the Y Axis Minimum and Y Axis Maximum values for any or all of the output graphs in the Graph Limits Dialog box by directly typing a number or using the spin boxes next the output. If you type a number, you must click or tab outside the box to change the number.

BehavePlus normally determines minimum and maximum Y axis limits for each graph. These recommended values are initially displayed in the Y Axis Minimum and Y Axis Maximum entry fields. You may leave these as is, or modify them, for example, to produce a series of runs with graphs showing the same scale.

Click Ok to use the Minimum and Maximum values displayed in the entry fields (whether or not you have changed them). Clicking Cancel will ignore values you have entered and the predetermined Y Axis Limits will be used for all graphs. Once you have made a selection, the graphs will be displayed.

More options can be found by going to Configure > Appearance preferences.
