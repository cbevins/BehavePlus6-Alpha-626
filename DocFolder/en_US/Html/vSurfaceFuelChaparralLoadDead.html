Chaparral Total Dead Fuel Load

Chaparral total dead fuel load is the sum of the load of all dead fuel classes for the chaparral special case fuel model. The model developed by Rothermel and Philpot (1973) and modified by Cohen (1986) includes four size classes. The chaparral total dead fuel load is calculated from the chaparral dead fuel portion and the chaparral total fuel load.

I/O Module If Notes
Input None    
Output SURFACE If Fuel is entered as special case fuel model, chaparral is selected as an input option.  

Once the chaparral total dead fuel load is calculated, it is then partitioned as follows, where w is the total chaparral fuel load and f is the dead fuel fraction. The fuel load for each size class of dead fuel is a separate output.
