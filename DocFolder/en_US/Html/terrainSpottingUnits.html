Terrain & Spotting Units

This page controls the selection of units of measure for all Terrain and Spotting Distance variables. The number of decimal places displayed for output variables can be changed. The number of decimal places displayed for input variables is determined by the precision of user input, not by this page (a change made in version 5.0).

The units of measure initially displayed by the dialog are those in use for the current worksheet. The selected units and decimals are applied to the current worksheet and can also be saved as a custom Units Set.

To apply a previously saved custom Units Set to any BehavePlus worksheet, choose Configure > Units set selection > Custom from the menu and select the desired file.
