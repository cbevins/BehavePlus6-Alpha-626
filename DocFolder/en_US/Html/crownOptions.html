CROWN Input Options

This page controls the CROWN Module's input options. The first option for calculating crown fire behavior always applies. The surface fireline intensity option is meaningful only if CROWN is run without SURFACE.

Options

Crown fire is calculated using....
  Rothermel (1991). Rothermel's method for calculating fire behavior for active crown fires is used. Passive crown fire rate of spread is assumed to be the same as surface fire rate of spread.
Scott and Reinhardt (2001). Scott and Reinhardt's method for calculating fire behavior for both active and passive crown fires is used. Active crown fire behavior is the same as for Rothermel (1991). Passive crown fire rate of spread is assumed to be greater than the surface fire rate of spread and less than the active crown fire rate of spread. Passive crown fire behavior is based on the canopy fraction burned.
 
Surface fire intensity is entered as...
  flame length. If CROWN is run without SURFACE, flame length is entered and the program will determine the equivalent fireline intensity.
  fireline intensity. If CROWN is run without SURFACE, surface fireline intensity is entered.
