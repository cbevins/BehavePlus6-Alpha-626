Chaparral Total Live Fuel Load

Chaparral total live fuel load is the sum of the load of all live fuel classes. For the chaparral special case fuel model (Rothermel and Philpot 1973; Cohen 1986), it includes five size classes (one for leaves and four for stems). The chaparral total live fuel load is calculated from the chaparral dead fuel fraction and the chaparral total fuel load.

I/O Module If Notes
Input None    
Output SURFACE If Fuel is entered as special case fuel model, chaparral is selected as an input option.  

Once the chaparral total live fuel load is calculated, it is then partitioned as follows, where w is the total chaparral fuel load and f is the live fuel fraction. The fuel load for each size class of live fuel is a separate output.
