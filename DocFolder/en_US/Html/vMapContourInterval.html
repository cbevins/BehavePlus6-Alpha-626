Contour Interval

The contour interval is the difference in elevation between adjacent topographic contours on a topographic map. It is used in the calculation of slope steepness.

I/O Module If Notes
Input SURFACE If Slope steepness is calculated from map measurements is selected as an input option.  
Output None
