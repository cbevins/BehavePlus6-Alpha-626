D.B.H.

D.B.H. is the abbreviation for diameter at breast high. It is the measurement of the diameter of a tree measured at about 4.5 feet (1.4 meters) from the ground. It is used to calculate bark thickness in MORTALITY and is part of the torching tree description in SPOT.

D.B.H. is also an input for estimating mortality as part of the western aspen special case fuel model in SURFACE.

I/O Module If Notes
Input MORTALITY    
SPOT    
SURFACE If Fuel is entered as special case fuel model, western aspen is selected as an input option.  
Output None
