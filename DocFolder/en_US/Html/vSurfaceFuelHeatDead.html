Dead Fuel Heat Content

Dead fuel heat content is the amount of heat energy contained in a unit of dead fuel, and is a fuel model parameter. The heat content for the standard fire behavior fuel models is 8,000 Btu / lb (18,622 kJ / kg).

I/O Module If Notes
Input SURFACE If Fuel is entered as fuel parameters is selected as an input option. If Fuel is entered as fuel models or a special case fuel model then the appropriate stored value is used.
Output None
