Flat Terrain Spotting Distance from Torching Trees

Flat terrain spotting distance from torching trees is the maximum spotting distance over flat terrain. This value is then adjusted for terrain. This is an intermediate value in the calculations useful for examining the model for spotting distance from torching trees.

I/O Module If Notes
Input None    
Output SPOT   Intermediate value
