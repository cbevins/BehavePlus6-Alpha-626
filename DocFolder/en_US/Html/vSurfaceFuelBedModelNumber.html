Fuel Model Number

Fuel models are described in a three-part naming scheme, including

  • fuel model code, which is a 1-7 character identifier such as 1 or gr1; and
  • fuel model number, which is an integer in the range 1-256.
  • fuel model name, which is a longer description of the fuel model.

The fuel model number, ranging from 1 to 256, is used for computer code and mapping applications. The fuel model code and number are identical for the original 13 fuel models described by Anderson (1982). The Scott and Burgan (2005) fuel models are grouped by fire-carrying type, such as 12x (grass-shrub) or 20x (slash-blowdown). Custom fuel models require a code and number so that they can be exported for use in the FARSITE format.

Some fuel model numbers are reserved for fire modeling system development. For information on fuel model numbers, refer to Table 1 in the following publication available on the BehavePlus publications page of www.firemodels.org. .
