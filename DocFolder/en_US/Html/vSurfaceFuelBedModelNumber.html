Fuel Model Number

Fuel models are described in a three-part naming scheme, including:

  • Fuel Model Code, which is a 1-7 character identifier such as 1 or gr1; and
  • Fuel Model Number, which is an integer in the range 1-256.
  • Fuel Model Name, which is a longer description of the fuel model.

Either the fuel model code or the fuel model number can be entered in BehavePlus.

Fuel model number, ranging from 1 to 256, is most often used for computer code and mapping applications. The fuel model code and number are identical for the original 13 fuel models described by Anderson (1982). The Scott and Burgan (2005) fuel models are grouped by fire-carrying type, such as 12x (grass-shrub) or 20x (slash-blowdown). Custom fuel models require a code and number so that they can be exported for use in spatial systems such as FARSITE and FlamMap.

Some fuel model numbers are reserved for fire modeling system development. BehavePlus will not allow you to use those numbers when developing custom fuel models. For information on fuel model numbers, refer to the table below, which is also in the following publication available on the BehavePlus publications page of www.frames.gov/behaveplus.

  • Scott, J.H.; Burgan, R.E. (2005) Standard fire behavior fuel models: A comprehensive set for use with Rothermel's surface fire spread model. General Technical Report RMRS-GTR-153. Fort Collins, CO: U.S. Department of Agriculture, Forest Service, Rocky Mountain Research Station. 80 p.
