Bark Thickness

Bark thickness is used to calculate tree mortality for some species. Bark thickness is calculated from mortality tree species and D.B.H.

Previous versions of BehavePlus (version 4 and earlier) allowed users to calculate the bark thickness. Changes in the species mortality equations in version 5 meant that bark thickness calculations were required; therefore, it is no longer available as an input.

I/O Module If Notes
Input None    
Output MORTALITY    

BehavePlus estimates bark thickness from the tree species and D.B.H. using equations from FOFEM 6.0 User Guide (Lutes, 2013).
