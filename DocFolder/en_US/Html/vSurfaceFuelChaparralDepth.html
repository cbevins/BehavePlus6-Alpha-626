Chaparral Fuel Bed Depth

Chaparral fuel bed depth is a required input for the chaparral special case fuel model (Rothermel and Philpot 1973; Cohen 1986). It represents the depth of the surface fuel, including the chaparral; it does not include the duff layer.

I/O Module If Notes
Input SURFACE If Fuel is entered as special case fuel model, chaparral is selected as an input option.  
Output None    

For more information on chaparral fuel model parameters, see Chaparral Details.
