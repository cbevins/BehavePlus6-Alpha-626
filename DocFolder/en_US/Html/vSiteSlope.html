Slope Steepness

Slope steepness is the maximum steepness of the slope on which a fire is burning. It is calculated as "rise over run" or Slope Elevation Change divided by Slope Horizontal Distance. It is expressed as either percent or degrees depending on the input option that is selected. This is not necessarily the slope steepness in the direction of spread.

I/O Module If Notes
Input SURFACE If Slope steepness is specified on the worksheet is selected as an input option.  
Output SURFACE If Slope steepness is calculated from map measurements is selected as an input option.  

Calculating Slope Steepness

  • Assume the Slope Elevation Change (rise) is 400 ft.
  • Assume the Slope Horizontal Distance (run) is 800 ft.
  • Slope Steepness = (400 / 800) × 100% = 50%, or
  • Slope Steepness = arctan(0.50) = 27 degrees

When determining the slope steepness of a hillside, the user should select two points on the area of interest. The line connecting these two points should be, as much as possible, perpendicular to the contour lines, starting and ending at a contour line.
