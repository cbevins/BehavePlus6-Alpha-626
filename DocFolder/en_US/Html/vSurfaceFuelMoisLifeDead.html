Dead Fuel Moisture

When moisture is entered by live and dead categories is selected, the same dead fuel moisture value is used for 1-h, 10-h, and 100-h fuels. In most cases this will give exactly the same result as the following relationships, which are sometimes used.

  • 10-h = 1-h +1
  • 100-h = 1-h + 2

I/O Module If Notes
Input SURFACE If Moisture is entered by dead and live categories is selected as an input option.  
Output None
