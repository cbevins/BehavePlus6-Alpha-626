Chaparral Details

The chaparral special case fuel model in BehavePlus can be used with two types of chaparral: chamise and mixed chaparral. Equations from Rothermel and Philpot (1973) as well as Cohen (1986) are used to create fuel inputs for live and dead fuel by size class to define the necessary surface fuel parameters.

The following parameters are constants for all chaparral fuel models.

  • Dead fuel moisture of extinction = 20%
  • Live fuel moisture of extinction = 65% for chamise and 74% for mixed chaparral.

The following parameters are required inputs for the model.

  • Chaparral Fuel Type
  • Chaparral Fuel Bed Depth
  • Chaparral Fuel Dead Load Fraction
  • Chaparral Total Fuel Load

Rothermel and Philpot (1973) originally allowed the user to calculate these parameters based on the age of the chaparral, but that has led to incorrect estimates in the model. Therefore, direct entry of the values is required. Additional fuel model inputs can be found in the table below.
