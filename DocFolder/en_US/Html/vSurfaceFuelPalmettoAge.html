P-G Age of Rough

Palmetto-gallberry age of rough (understory vegetation) is applicable only for the special case palmetto-gallberry fuel model. The age of understory rough is usually the number of years since the last fire; it is an important factor in the calculation of fuel load for all fuel classes.

I/O Module If Notes
Input SURFACE If Fuel is entered as special case fuel model, palmetto-gallberry is selected as an input option.  
Output None    

For more information on palmetto-gallberry fuel parameters, see Palmetto-Gallberry Details.
