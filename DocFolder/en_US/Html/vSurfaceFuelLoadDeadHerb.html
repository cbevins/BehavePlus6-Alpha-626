Dead Herbaceous Fuel Load

Dead herbaceous fuel load is the weight of dead herbaceous fuel per unit area and applies only to dynamic standard and custom fuel models. It is not the 1-h fuel load. The initial value is zero. A portion of the live herbaceous fuel load may be transferred into the dead herbaceous fuel load category. The portion of the load transferred is either specified by the user or calculated from live herbaceous fuel moisture content. The moisture content for this fuel category is the value assigned to the 1-h fuel. The surface area-to-volume ratio is the value for the live herbaceous fuel.

See the Fuel Model Type for specifics about the load transfer equation.

I/O Module If Notes
Input None    
Output SURFACE If the standard or custom fuel model is dynamic.
