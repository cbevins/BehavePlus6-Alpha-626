Live Herbaceous Moisture

Live herbaceous fuel moisture content is the portion of the live grasses and forbs that is water, calculated on a dry weight basis. This can be a very important value for dynamic fuel models if it is used to determine the amount of fuel that is transferred from the live herbaceous fuel load category to dead herbaceous fuel load. (Alternatively, the user can directly specify the portion of fuel load that is transferred, fuel load transfer portion.)

I/O

Module

If

Notes
Input SURFACE If live herbaceous fuels are in the fuel model.

If the fuel model is dynamic and if Dynamic curing load transfer is calculated from live herbaceous fuel moisture is selected, the value is used to determine live and dead herbaceous load values.

If the Fuel is entered as special case fuel model, palmetto-gallberry is selected, the value is used for P-G live foliage moisture.

Output None    

Following is a rough guideline for estimating live fuel moisture content when no other information is available.

Guidelines for Estimating Live Fuel Moisture Content
Moisture Stage of Vegetative Development
300% Fresh foliage, annuals developing, early in growing cycle
200% Maturing foliage, still developing with full turgor
100% Mature foliage, new growth complete and comparable to older perennial foliage
50% Entering dormancy, coloration starting, some leaves may have dropped from the stem
30% Completely cured, treat as dead fuel
