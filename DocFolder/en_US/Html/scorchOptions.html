SCORCH Input Options

This page controls the SCORCH Module's input options.

While scorch height is normally calculated from flame length, the SCORCH Module also permits direct entry of Byram's fireline intensity. This option has an effect only if SCORCH is run without SURFACE.

Options

Surface fire intensity is entered as....
  flame length.

If SCORCH is run without SURFACE, flame length is entered and the program will determine the equivalent fireline intensity.

  fireline intensity.

If SCORCH is run without SURFACE, fireline intensity is entered.
