Surface Fire Maximum Effective Wind Exceeded?

Surface fire maximum effective wind exceeded? has a value of Yes or No to indicate whether the effective wind speed limit, as calculated in the surface fire spread model (Rothermel 1972), has been exceeded. Andrews, Cruz, and Rothermel (2013) show that the wind limit is too restrictive and recommend that it not be imposed. In the rare case where head fire rate of spread is greater than effective wind speed (for effective wind > 0), surface fire rate of spread is set to the effective wind speed.

The user has the option to Impose maximum effective wind speed limit? (Yes or No). If Yes, the value for effective wind speed limit is used in the calculations. If No, the input values are used. For the aspen special case fuel model, the wind limit is never applied.

I/O Module If Notes
Input None    
Output SURFACE   An output value even if the wind limit is not imposed.
