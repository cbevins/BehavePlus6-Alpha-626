Aspen Fuel Curing Level

Aspen fuel curing level is the percent (or fraction) of herbaceous biomass that is dead. This value is used only for the western aspen special case fuel models (Brown and Simmerman 1986). It is used to calculate several aspen fuel parameters.

  • 1-h fuel load
  • 1-h fuel surface area-to-volume ratio
  • live herbaceous fuel load
  • live woody fuel load
  • live woody fuel surface area-to-volume ratio

Aspen fuel curing level is different from the fuel load transfer portion used to represent curing for dynamic fuel models (standard or custom) (Scott and Burgan 2005).

For more information on aspen fuel parameters, see Western Aspen Details.

I/O Module If Notes
Input SURFACE If Fuel is entered as special case fuel model, western aspen is selected as an input option  
Output None
