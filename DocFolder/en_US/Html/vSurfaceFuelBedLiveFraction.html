Live Fuel Load Portion

Live fuel load portion is the percent or fraction of the total fuel load that is live fuel. The calculation is made after the load transfer is done for dynamic fuel models, and is most useful when examining these fuel models.

I/O Module If Notes
Input None    
Output SURFACE   Of primary interest for dynamic fuel models
