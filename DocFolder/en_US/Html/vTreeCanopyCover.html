Canopy Cover

Canopy cover is the percent of the ground covered by overstory canopy. It is used to calculate wind adjustment factor.

I/O

Module

If

Notes
Input SURFACE If Wind speed is entered as 20-ft (or 10-m) wind and Calculated wind adjustment factor is selected as an input option.  
Output None
