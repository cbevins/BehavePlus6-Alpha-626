Worksheet Appearance Options

This page controls the amount of additional input documentation that appears on the BehavePlus input worksheet.

  • Prescribed fire plan worksheet header
    Checking this item causes the following input fields to appear on the worksheet:
    • Administrative Unit
    • Prescribed Fire Name
    • Prepared By

  • Fire behavior projection worksheet header
    Checking this item causes the following input fields to appear on the worksheet:
    • Fire Name
    • Fire Date & Projection Period
    • Fire Analyst

  • Training worksheet header
    Checking this item causes the following input fields to appear on the worksheet:
    • Training Course
    • Training Exercise
    • Trainee Name and Date

  • Show descriptions for all discrete variable codes
    Checking this item causes the valid input codes and descriptions of all discrete input variables appearing on the worksheet to be displayed on the worksheet. If, for example, Fuel Model is an input item, then checking this option will cause all the codes and descriptions of all attached fuel models (a minimum of 53) used in the SURFACE module to be listed on the worksheet. This option is rarely used.

  • Show descriptions only for entered discrete variable codes
    Checking this item causes the valid codes and descriptions of only those items entered for discrete input variables appearing on the worksheet to be displayed on the first output page. If, for example, Fuel Model is an input item and has 'gr1' and 'sh1' entered as input, then checking this option will cause all the codes and descriptions for these two fuel models to be listed on the last output page.

  • Show output variables to be calculated
    Checking this item causes a list of all currently selected output variables and their units of measure to be displayed at the end of the worksheet.

  • Show notes section
    Checking this item causes a small notepad to appear at the end of the worksheet in which documentation may be typed. The notepad is a simple editor with automatic word wrap. Any text entered in the notepad appears on the printed worksheet.

  • Note Lines
    This spin box sets the maximum number of lines displayed in the Notes section of the worksheet and saved in the run file.
