Flame Height from a Burning Pile

Flame height from a burning pile is the average height of the flames measured from the base of a burning pile. This is not the same as flame length from a spreading surface fire. There is no model to calculate flame height from a burning pile; it is an input determined by expert opinion.

I/O Module If Notes
Input SPOT If Spotting distance from a burning pile is selected as an output variable.  
Output None
