Firebrand Height from Active Crown Fire

The firebrand height from active crown fire is the maximum height to which a firebrand is lifted and is still viable for starting a fire when it lands. This is an intermediate value in the calculations that might be of interest to someone examining the active crown fire spotting distance model.
