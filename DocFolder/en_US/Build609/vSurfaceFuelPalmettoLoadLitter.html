P-G Litter Fuel Load

The P-G litter fuel load for the palmetto-gallberry fuel model is the load of the litter layer. This variable is calculated from P-G age of rough and P-G overstory basal area.

I/O

Module

If

Notes
Input None    
Output SURFACE If Fuel is entered as special case fuel model, palmetto-gallberry is selected.  

For more information on palmetto-gallberry fuel parameters, see Palmetto-Gallberry Details.
