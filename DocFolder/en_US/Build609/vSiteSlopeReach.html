Slope Horizontal Distance

The slope horizontal distance is the ground distance between two points. This is an intermediate value in the calculation of slope steepness from map measurements.

I/O

Module

If

Notes
Input None    
Output SURFACE If Slope steepness is calculated from map measurements is selected as an input option.
