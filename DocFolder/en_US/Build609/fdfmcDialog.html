Fine Dead Fuel Moisture Tables

The fine dead fuel moisture calculation is based on the tables in How to predict the spread and intensity of forest and range fires (Rothermel 1983). Only the daytime tables are provided, and there is no conversion to metric units.

This tool is included in BehavePlus only for completeness because the tables are used in the field by Fire Behavior Analysts. They are not based on equations, so they are not readily linked with other modules in BehavePlus.

Fine dead fuel moisture values can be used for 1-h and 10-h fuel moisture.

You may save the current dialog values in an HTML file by pressing the Export button. A file selection dialog allows you to save the contents in a new or existing file.

Fine dead fuel moisture content is the sum of a reference fuel moisture and a moisture correction.

The reference fuel moisture is a function of the dry bulb temperature and relative humidity, which are determined for the period of time in question either by measurement or forecast. The site location may or may not be at the projection point.

The daytime fuel moisture correction value is a function of month, time of day, elevation difference, slope steepness, aspect, and shading. The dependence on month limits use of the tables to the mid-latitudes of the northern hemisphere.

Time of day is the expected time when the projection point is used to estimate fire behavior; it is the same as that for the temperature and relative humidity estimates.

Elevation difference is the difference between the projection point and the temperature/RH reference locations. If the projection point is more than 2000 ft above or below the temperature/RH reference site, you will need to get a new forecast or reading.

Slope steepness and aspect are for the projection point location.

Shading indicates whether or not the fine dead fuels at the projection point are exposed to solar radiation. Shading can be due to cloud cover, topography, and/or canopy cover.

Description is optional description text that is included in the exported HTML file.
