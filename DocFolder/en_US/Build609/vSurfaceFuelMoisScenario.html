Moisture Scenario

A fuel moisture scenario is a set of fuel moistures values representing moisture condition of the surface fuel: 1-h, 10-h, 100-h, live herbaceous, and live woody moisture. Fuel moisture scenarios are analogous to fuel models; they package a number of parameters into a labeled set. The user can define and save a moisture scenario for later use.

Fuel moisture scenarios are useful for testing custom fuel models or for developing fire prescriptions. For example, a moisture scenario might be developed to represent a location's 90th-, 95th-, and 97th-percentile fire weather conditions.

Fuel moisture scenarios are useful for testing custom fuel models. For example, moisture scenarios representing a location's 90th-, 95th-, and 97th-percentile fire weather conditions can be used to test a candidate fuel model's performance. All three scenarios may be entered on the input worksheet to generate comparative tables and graphs (not possible using fuel moisture by size class).

To use a fuel moisture scenario, enter its name in the input field or press the guide button to select scenarios from the list.

Custom moisture scenarios can be developed and used within BehavePlus. To create a custom moisture scenario, click on Configure -> Modules selection and press the Surface Fire Spread Options button. Under the Fuel & Moisture tab, activate the radio button titled Moisture is entered by... individual size class. The resulting input worksheet requests all the parameters needed to define a custom moisture scenario.

To save the values as a custom moisture scenario, press File -> Save As -> Moisture scenario.

Before custom moisture scenarios can be used by BehavePlus, they must be attached to the run via Configure -> Moisture scenario set selection. The attached moisture scenarios will then be accepted as valid inputs to the worksheet and will appear in the list under the Guide button.

To view the parameters of a particular moisture scenario, right-click on its name and select View parameters from the context menu.
