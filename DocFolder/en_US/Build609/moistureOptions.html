SURFACE Moisture Input Options

This page controls the input of moisture parameters into the SURFACE Module.

Options

Dynamic curing load transfer is....
  calculated from live herbaceous fuel moisture.

'Dynamic' fuel models include the transfer of live herbaceous fuel load to the dead category as a function of live herbaceous fuel moisture (Scott and Burgan 2005). Seventeen of the 40 new fuel models are dynamic.

  input directly.

Rather than calculating the load transfer portion from live fuel moisture, it can be entered directly. This allows examination of the influence of the load transfer function on calculated fire behavior (Jolly 2005, Andrews 2006).

 
Moisture is entered by....
  individual size class.

The moisture content is entered for each fuel size class 1-h, 10-h, and 100-h dead fuels and herbaceous and woody live fuels.

dead and live categories.

The moisture content value entered for 'dead' is used for 1-h, 10-h, and 100-h fuel. The moisture content for 'live' is used for live herbaceous and live woody fuel. This does not include foliar moisture (conifer needles).

dead and live herb, and live woody categories.

The moisture content value entered for 'dead' is used for 1-h, 10-h, and 100-h fuel. The moisture content for live herbaceous and live woody fuel are entered separately. This does not include foliar moisture (conifer needles).

moisture scenario.

A moisture scenario is a predefined set of fuel moistures for 1-h, 10-h, and 100-h dead fuel and herbaceous and woody live fuel. It is analogous to the fuel model concept in that a single code represents a set of fuel. Fuel moisture scenarios may be developed, for example, to represent local 90-, 95-, and 97-percentile weather situations.
