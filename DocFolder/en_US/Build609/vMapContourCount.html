Number of Contour Intervals

The number of contour intervals is a count of the number of contour intervals between two points on a map. It is used in the calculation of slope steepness.

When determining the slope steepness of a hillside, the user should select two points with one at the bottom of the slope and the other at the top. The line connecting these two points should be, as much as possible, perpendicular to the contour lines.

I/O

Module

If

Notes

Input

 SURFACE

If Slope steepness is calculated from map measurements is selected as an input option.

  

Output

None

 

 

The slope steepness (percent) between two points is calculated as 100 times the vertical distance between the points divided by the horizontal distance between the points. To get the vertical distance between the two points, one must count the number of contour intervals on the topographic map that intersect a line drawn between the two points. The vertical distance between the two points is the number of contour intervals times the vertical distance between adjacent contour lines.
