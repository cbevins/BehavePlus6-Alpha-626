Direction of Wind Vector (from upslope)

Direction of the wind vector measured in degrees from upslope is the direction that the wind is pushing the fire. This is useful in examining the vectoring approach to wind and slope influences.

Alternatively the user can select the option to specify wind direction as the direction from which the wind is blowing. It is obviously critical that the user be certain of the method of specification of wind direction.

I/O

Module

If

Notes
Input SURFACE If Wind is in specified directions
and if Wind and spread directions are degrees clockwise from upslope (direction the wind is pushing the fire) are selected as input options.		  
Output None
