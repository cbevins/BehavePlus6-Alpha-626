BehavePlus Help

Users of the BehavePlus fire modeling system are expected to have prior training and experience with fire modeling. A user is expected to have basic knowledge of fire and fuels and influencing factors.

  • The User's Guide (PDF) is available from the Help menu. This gives guidance on program operation. The version 4.0 User's Guide remains valid and, therefore, it will not be updated.
  • A Variable Help (PDF) document is available from the Help menu. It includes input and output tables and a description of each variable.
  • A help window is available during program operation to provide context-sensitive help.
  • A guide button next to each worksheet entry provides a description of the variable.
  • Holding the cursor over variables in the output selection list provides a description of the variable.
  • A Master Index of variables, figures, tables, and guides is available from the Help menu and from the Help window.
  • Tutorials and other supporting documentation are available on www.firemodels.org.
BehavePlus On-Line Documentation, June 26, 2009.
