Bark Thickness

Bark thickness is used to calculate tree mortality for some species. Bark thickness is calculated from mortality tree species and D.B.H.

I/O

Module

If

Notes
Input None    
Output MORTALITY    

BehavePlus estimates bark thickness from the tree species and D.B.H. using equations from FOFEM 6.0 User Guide (Lutes, Duncan C., 2013).

Each tree species has a bark thickness equation and a fire mortality equation associated with it. These are summarized in the table:
