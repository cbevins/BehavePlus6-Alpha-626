CROWN Input Options

This page controls the CROWN Module's input options. This option is meaningful only if CROWN is run without SURFACE.

Options

Surface fire intensity is entered as...
  flame length. If CROWN is run without SURFACE, flame length is entered and the program will determine the equivalent fireline intensity.
  fireline intensity. If CROWN is run without SURFACE, fireline intensity is entered.
