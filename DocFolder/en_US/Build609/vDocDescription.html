Description

A brief phrase documenting the run. The description is used to identify the file when it is saved and opened. The run description appears at the top of all output pages and is stored with the Run file.

Other lines can be added to the worksheet header by selecting Configure > Appearance preferences > Worksheet and checking the Prescribed fire plan worksheet header, Fire behavior projection worksheet header, and/or Training worksheet header check boxes.

Additional description can be added in the Notes section.
