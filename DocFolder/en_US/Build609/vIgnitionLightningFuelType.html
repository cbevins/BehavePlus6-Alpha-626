Lightning Ignition Fuel Type

Lightning ignition fuel type is used to calculate probability of ignition from a lightning strike. There are eight fuel types, which are only used for this ignition model.

I/O

Module

If

Notes
Input IGNITE   For probability of ignition by lightning
Output None    

Depending on fuel type, ignition from lightning is a function of either 100-h moisture content or lightning duff and litter depth.
