Active Crown Canopy Height

Active crown canopy height is the distance from the ground to the top of a tree. It is the height used to calculate maximum spotting distance from an active crown fire in the SPOT module.

I/O

Module

If

Notes
Input SPOT If Spotting distance from active crown fire is selected as an output variable.  
Output None
