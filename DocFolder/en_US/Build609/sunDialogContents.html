Sun-Moon Calendar Contents

This page defines the contents of the Sun-Moon calendar.

Controls

New/Full/ Moons and Seasons Check this box to produce a table of new and full moon dates and times for the entire year.
The table also contains the year's solstice and equinox dates and times.
Sun-Moon Chart

Check this box to produce a table of daily sun and moon events for the month.
Content options include:

  • Sunrise
       Check this box to have sunrise times appear in the Sun-Moon chart.
  • Sunset
       Check this box to have sunset times appear in the Sun-Moon chart.
  • Moonrise
       Check this box to have moonrise times appear in the Sun-Moon chart.
  • Moonset
       Check this box to have moonset times appear in the Sun-Moon chart.
  • Civil Dawn
  • Civil Dusk
Sun-Moon Calendar Check this box to produce a calendar for the specified month.
Content options include:
  • Sun Times
       Check this box to have sunrise and set times appear in each day on the calendar.
  • Moon Times
       Check this box to have moonrise and set times appear in each day on the calendar.
  • Moon Phase
       Check this box to mark the calendar with the days of the new moon, the full moon, and the first and third quarters.
