10-h Fuel Load

10-h fuel load is the weight of the 10-h dead fuel per unit area, and is a fuel model parameter. The ten-hour (10-h) timelag dead fuel category includes fuels from 0.25 to 1.0 inch (0.64 to 2.54 cm) in diameter.

I/O

Module

If

Notes
Input SURFACE If Fuel is entered as fuel parameters is selected as an input option. If Fuel is entered as fuel models or a special case fuel model aspen then the appropriate stored value is used.
