Dead Fuel Moisture

The dead fuel moisture value is used for 1-h, 10-h, and 100-h fuels. In most cases this gives exactly the same result as the sometimes used relationships:

  • 10-h = 1-h +1 and
  • 100-h = 1-h + 2.

I/O

Module

If

Notes
Input SURFACE If Moisture is entered by dead and live category is selected as an input option.
