Module Selection

Check those Modules you wish to activate.

Indentation indicates modules that can be linked together, with the output from one used as input to the next.

Press a Module's Options... button to configure its input options or change the set of output variables it calculates.

Checking the box Display output distances in map units adds map descriptors to the worksheet so output variables such as "spread distance" are displayed in both real world and map units. Checking the box Table shading for acceptable fire conditions adds a section to the worksheet where values for acceptable conditions can be entered (for example, flame length of 2 to 4 ft).
