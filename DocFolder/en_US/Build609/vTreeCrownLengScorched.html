Tree Crown Length Scorched

Tree crown length scorched is calculated as

  • scorch height - (canopy height - live crown length).
If scorch height is greater than the canopy height, then tree crown length scorched is set equal to the tree crown length. Tree crown length scorched is used to calculate percent tree crown volume scorched, which is used to calculate tree mortality for some species.
